# Antigravity 全平台汉化包 (macOS & Windows)

本项目为 Google Antigravity 桌面客户端提供一键式中文汉化支持。无论你是 macOS 还是 Windows 用户，都可以轻松完成安装。

与原版相比，本优化版抛弃了直接替换底层文件导致的降级风险，改为**安全的动态注入**，并支持了双平台。此外，我们加入了代码区域隔离机制，确保翻译不干扰你正常的代码编写，同时大幅提升了软件运行时的流畅度。

---

## 🚀 快速上手

### ⚠️ 必备环境
请确保您的电脑上已安装 **Node.js**（汉化过程需要用到其解包工具）。
- 如未安装，请先前往 [Node.js 官网](https://nodejs.org) 下载并安装。

### 🍏 macOS 用户安装指南
1. 下载并解压本项目。
2. 找到并双击运行 **`macOS汉化工具.app`**。
3. 在弹出的窗口中点击 **“安装汉化”**。
4. 稍等片刻，客户端会自动重启并呈现中文界面。

### 🪟 Windows 用户安装指南
1. 下载并解压本项目。
2. 找到并双击运行 **`Windows汉化补丁.bat`**。
3. 在弹出的控制台窗口中，输入数字 `1` 并回车。
4. 等待提示成功，按回车退出即可。

---

## 🛡️ 如何卸载 / 恢复英文原版？

安装工具会自动备份你的官方文件。如果你想随时恢复英文原版：
- **macOS**：再次双击 `macOS汉化工具.app`，选择 **“卸载还原”**。
- **Windows**：再次双击 `Windows汉化补丁.bat`，输入数字 `2` (恢复原版) 并回车。

一键操作即可完美恢复纯正官方英文环境。

## 💡 常见问题 (FAQ)

### Q：汉化完成后，可以删除本汉化包文件夹吗？
**A：完全可以。**
汉化补丁代码在安装时已安全写入软件内部。**如果你觉得这个文件夹放在桌面太碍眼，直接删掉即可**，不会影响软件的中文显示。
*（小提示：如果以后官方软件升级覆盖了汉化，或者你想一键还原英文，需要重新下载本汉化包运行一次。）*

## ⚠️ 注意事项与已知瑕疵
1. **不完全翻译**：由于采用非破坏性的动态注入机制，少部分界面文案（或动态生成的提示）可能没有被翻译，仍会显示为英文，这属于正常现象。
2. **测试状态**：目前 **macOS 版本已验证运行正常**。Windows 版本的程序脚本（`.bat` 与 PowerShell）虽然已经重写并合入本项目，但**尚未经过真机可行性验证**，可能存在未知问题，欢迎 Windows 用户在使用后提交反馈。
