# Antigravity 全平台汉化包 (macOS & Windows)

本项目为 Google Antigravity 桌面客户端提供一键式中文汉化支持。无论你是 macOS 还是 Windows 用户，都可以轻松完成安装。

与原版相比，本优化版抛弃了直接替换底层大文件导致的降级风险，改为**安全的细粒度动态注入**（完美保留源程序完整性），并全面支持了双平台原生图形交互。此外，我们加入了代码区域隔离机制，确保翻译不干扰你正常的代码编写，同时大幅提升了软件运行时的流畅度。

---

## 🚀 快速上手

### ⚠️ 必备环境
请确保您的电脑上已安装 **Node.js**（汉化过程需要用到其解包工具）。
- 如未安装，请先前往 [Node.js 官网](https://nodejs.org) 下载并安装。

### 🍏 macOS 用户安装指南
1. 下载并解压本项目。
2. 进入 `macOS` 文件夹。
3. 双击运行 **`macOS汉化工具.app`**。
4. 在弹出的窗口中点击 **“安装 / 更新汉化”**。
5. 稍等片刻，客户端会自动重启并呈现中文界面。

### 🪟 Windows 用户安装指南
1. 下载并解压本项目。
2. 进入 `Windows` 文件夹。
3. 双击运行 **`windows汉化工具.exe`**。（这是纯原生程序，无任何黑框闪烁烦恼）
4. 在弹出的图形界面窗口中，点击 **“安装 / 更新汉化”**。
5. 稍等片刻，等待弹窗提示成功即可，客户端会自动重启呈现中文界面。
*(如果您依然喜欢极客命令行，也可以运行 `patch.bat` 使用命令行版本)*

---

## 🛡️ 如何卸载 / 恢复英文原版？

安装工具会自动对您的原始核心文件进行微型备份。如果你想随时恢复官方原生纯净版本：
- **macOS**：再次双击运行 `macOS汉化工具.app`，选择 **“卸载并还原官方原版”**。
- **Windows**：再次双击运行 `windows汉化工具.exe`，点击 **“卸载并还原官方原版”**。

一键操作即可完美恢复纯正官方英文环境。

## 📁 项目目录结构与分发说明

本项目的重构架构支持极简的独立打包分发，您可以直接将 `Windows` 或 `macOS` 目录单独打成 zip 压缩包分享，两个平台相互完全独立：

```text
Antigravity-Chinese-Patch/
 ├── Windows/
 │    ├── windows汉化工具.exe      (Windows 用户专属原生 GUI)
 │    ├── patch.bat                (经典命令行版)
 │    ├── lib/                     (核心 PowerShell 脚本库)
 │    └── dist/                    (核心汉化注入依赖代码)
 │
 ├── macOS/
 │    ├── macOS汉化工具.app        (完全独立绿色的 Mac 应用，支持直接分享)
 │    ├── install.sh               (极客备用命令行脚本)
 │    └── macOS汉化工具.applescript (源码)
 │
 ├── README.md
 └── LICENSE
```

## 💡 常见问题 (FAQ)

### Q：汉化完成后，可以删除本汉化包文件夹吗？
**A：完全可以。**
汉化补丁代码在安装时已安全写入软件内部。**如果你觉得这个文件夹放在桌面太碍眼，直接删掉即可**，不会影响软件的中文显示。
*（小提示：如果以后官方软件升级覆盖了汉化，或者你想一键还原英文，需要重新下载本汉化包运行一次还原功能。）*

## ⚠️ 注意事项与已知瑕疵
- **不完全翻译**：由于采用非破坏性的动态注入机制，少部分界面文案（或动态生成的提示）可能没有被翻译，仍会显示为英文，这属于正常现象。
